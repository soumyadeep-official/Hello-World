# Hello_World in C

A simple **Hello, World!** program written in the C programming language. This project is intended for beginners who are learning the basics of C programming and setting up their development environment.

## 📌 About

This program demonstrates the basic structure of a C program, including:

* Importing the standard input/output library
* Defining the `main()` function
* Using `printf()` to display text on the console
* Returning an exit status

## 📂 Project Structure

```text
Hello_World/
├── hello.c
└── README.md
```

## 💻 Code

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

## 🚀 How to Compile and Run

### Using GCC

1. Compile the program:

```bash
gcc hello.c -o hello
```

2. Run the executable:

**Linux/macOS**

```bash
./hello
```

**Windows**

```bash
hello.exe
```

## 📤 Output

```text
Hello, World!
```
